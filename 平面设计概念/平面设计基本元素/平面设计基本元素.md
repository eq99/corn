# 平面设计基本元素

# 线 (line)

线能是连接两个点的形状。

线条有粗有细。

线条可能是直的，波浪形的，或锯齿等形状的，每种风格的线条给人不同的感觉。

线条是设计的基本组成元素，例如文字和图案。

线条和文字组合在一起表示强调，分割或组织内容，还能起到引导视觉的作用。

线条要考虑的要素是：粗细 (weight)，颜色 (color)，纹理 (texture)，风格 (style)。
