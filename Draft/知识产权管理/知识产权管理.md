# 著作权的客体

第一节：作品的概念和分类

作品是文学、艺术和科学领域内具有独创性并能以某种形式复制的智力成果。

作品是思想的表达。著作权保护的客体是对思想的表达，而非思想保护。

作品需要具有独创性。著作权法对作品的独创性要求并不高，远低于专利法对发明的创造性要求。

# 专利权的特点

- 专利权具有独占性
- 专利权的客体具有公开性
- 专利权具有法定授权性
- 专利权的效力具有局限性：（1）地域局限性，（2）时间局限性，（3）在某时候，未经权利人同意，并未支付报酬的行为的行为法律不认为侵权。



## 专利法

专利法调整的首先是发明成果的的所有权关系。

专利法调整发明创造的流转而产生的财产关系。

专利法还调整因发明创造而引起的人身关系。



### 作用

- 激励创造，推动科技进步
- 创造无形财产，提升利益资源
- 利用专利文献，提高创新起点
- 保护投资，提升市场竞争力
- 先入为主，防御竞争对手
